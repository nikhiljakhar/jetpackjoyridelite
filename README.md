# How to open
1. Download or clone this repository.
2. cd /path/to/this/repository
3. cd build
4. ./graphics_asgn1
 
# CONTROLS
1. LEFT KEY = move left
2. RIGHT KEY = move right
3. UP KEY = jump
4. SPACE = jetpack
5. KEY B = throw balloon
6. scroll up = zoom in
7. scroll down = zoom out
8. left mouse click = return screen to orignal size
